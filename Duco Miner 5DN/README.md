# Duco Miner 5DN
Arduino Nano (ATMEGA328) and Wemos D1 (ESP8266) based DUCO/Duino Coin miner. Uses a Wemos D1 as an I2C master controlling the 5 Nano I2C slaves. WiFi/Network handled by the Wemos microcontroller.
